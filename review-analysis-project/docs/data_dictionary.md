 Data Dictionary

| Column Name     | Description |
|----------------|------------|
| rating         | Product rating (1–5) |
| title          | Review title |
| review_text    | Customer review text |
| cleaned_text   | Processed text for analysis |
| asin           | Product identifier |
| helpful        | Number of helpful votes |
| verified       | Verified purchase flag (True/False) |
| date           | Review timestamp |
| sentiment      | Derived sentiment (Positive/Neutral/Negative) |
| review_length  | Word count of review |
